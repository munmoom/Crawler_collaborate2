프로젝트 내용 :

네이버 쇼핑' 이라는 사이트에서 리뷰 많은순의 상품 리스트를 크롤링하며 검색된 상품에 따라 나오는 각각의 상품명, 카테고리, 가격을 '리뷰 많은순'으로 표로 보여주는 프로그램 작성. (검색하는 상품은 '커피'로 고정됨.)
